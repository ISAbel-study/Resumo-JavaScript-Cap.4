Introdução a Programação em JavaScript

Conceito

O JavaScript é uma linguagem de programação usada para criar recursos dinâmicos em sites. O nome oficial que dita as regras e atualizações da linguagem é ECMAScript, e ele ganha novidades todo ano. Existem duas formas principais de usar o JavaScript em uma página web:

  • O jeito recomendado: Salvar o código em um arquivo separado (com final .js) e chamá-lo dentro do seu arquivo HTML usando o comando: .
  • Direto no HTML: Colocar o código direto na página usando a tag script.

A tag noscript: Serve para mostrar uma mensagem de aviso caso o navegador do usuário esteja com o JavaScript desativado.

Para fazer testes rápidos e ver se o código está funcionando, você pode usar dois caminhos:

1. Caixas de Alerta (Aparecem na tela do usuário)

  • alert(): Mostra um balão com uma mensagem e um botão "OK".
  • confirm(): Mostra uma mensagem com os botões "OK" e "Cancelar".
  • prompt(): Abre uma caixinha para o usuário digitar algum texto.

Nota: Como essas caixinhas travam a tela e podem incomodar o usuário, evite usá-las no site final.

2. O Console do Desenvolvedor (O jeito ideal)

É uma ferramenta "escondida" no navegador, perfeita para os programadores testarem códigos de forma silenciosa, sem atrapalhar quem está navegando. Os principais comandos são:

  • console.log() ou console.info(): Envia mensagens comuns de texto.
  • console.debug(): Usado para mensagens de teste mais detalhadas.
  • console.warn(): Mostra um aviso de atenção (geralmente fica amarelo).
  • console.error(): Mostra um erro grave (geralmente fica vermelho).

Variáveis, Constantes, Tipos de Dados e Operadores em JavaScript

Definindo Variáveis, Constantes
Variáveis (let e var): São caixinhas onde o valor pode mudar ao longo do tempo. No JavaScript moderno, variáveis são criadas usando a palavra-chave let. Você cria a variável e depois dá um valor para ela. Também pode encontrar códigos usando var. É uma forma mais antiga de criar variáveis que funciona de um jeito parecido, mas hoje em dia o let é o mais recomendado.
Constantes (const): São caixinhas onde o valor nunca muda. Uma vez que você coloca um dado lá dentro, ele fica fixo até o programa terminar. Constantes são criadas usando a palavra-chave const. É uma convenção entre programadores escrever o nome das constantes todo em letras maiúsculas (exemplo: PI ou URL_DO_SITE). Boa prática: A regra de ouro no JavaScript é: usar sempre const para tudo.

Usando Tipos de Dados
Números (number): Servem para fazer contas. No JavaScript, números inteiros (como 5) e números com vírgula (como 5.5) são tratados do mesmo jeito.
Textos (string): Qualquer texto que você colocar entre aspas simples ('text') ou aspas duplas ("text"). Também existem as template strings, que usam crases (`text`) e deixam você encaixar variáveis direto no meio do texto usando ${variável}.
Booleanos (boolean): São valores de verdadeiro (true) ou falso (false), muito usados para testar condições no código.
Valores vazios (null e undefined): Servem para indicar que uma variável não tem nenhum valor ou está totalmente vazia.
Símbolos (symbols): Um tipo especial do JavaScript, mas que não foi detalhado no capitulo 4.
Objetos e Arrays: Tipos mais complexos que servem para guardar vários valores juntos dentro de uma única variável.

Usando operadores

Os operadores são os símbolos que usamos para mexer, juntar e comparar os dados. Os principais são:

  • Operadores aritméticos (Matemática): Fazem as contas básicas: somar (`+`), subtrair (`-`), multiplicar (`*`) e dividir (`/`).
    • O `%` descobre o resto de uma divisão.
    • O `++` aumenta o valor em 1, e o `--` diminui em 1.
    • O `**` serve para fazer potências (ex: 2**3).
  • Operador de atribuição (`=`): Serve para guardar um valor dentro de uma variável.
  • Operador de concatenação (`+`): Quando você usa o sinal de mais entre dois textos, ele não soma, mas sim junta os textos em um só.
    • Exemplo: `"Olá, " + "mundo!"` vira `"Olá, mundo!"`.
  • Operadores lógicos e de comparação: Servem para testar condições. O operador `&&` (E) exige que duas condições sejam verdadeiras ao mesmo tempo; o `||` (OU) precisa que apenas uma seja verdadeira. Também existem os que comparam se um valor é igual ou diferente do outro.

Usando Estruturas de Controle

Usando Declarações Condicionais e Ramificação

As estruturas condicionais funcionam como desvios no trânsito: elas dizem ao computador qual caminho o código deve seguir dependendo do que for verdadeiro ou falso.

  • A instrução if: É a condição básica. O código que está dentro das chaves {} só vai rodar se o que estiver dentro dos parênteses () for verdade.
  • As ramificações else if e else: Servem para criar caminhos alternativos. O else if cria uma nova pergunta caso a anterior seja falsa. O else é o "plano B" definitivo: ele só roda se nenhuma das condições anteriores der certo.
  • O comando switch: É perfeito para substituir vários if e else longos quando você quer testar os diferentes valores de uma mesma variável. Ele olha o valor e vai direto para a opção correspondente (chamada de case).

Usando Loops (Laços de Repetição)

Os loops servem para repetir um bloco de código várias vezes de forma automática, enquanto uma condição for verdadeira. O JavaScript tem três tipos principais:

  • Loop de contagem for: É o mais usado quando você já sabe exatamente quantas vezes quer repetir o código. Ele usa uma variável para contar as repetições e, dentro dos parênteses, você define três coisas: onde a contagem começa, qual é o limite para ela parar e como ela deve aumentar (geralmente de 1 em 1).
  • Loop controlado pelo início while: Ele checa se a condição é verdadeira antes de rodar o código. Se a condição for falsa logo de cara, o código de dentro do loop não roda nenhuma vez.
  • Loop controlado pelo fim do-while: Ele faz o contrário do anterior: primeiro ele roda o código e depois checa a condição. Isso garante que o seu código vai rodar pelo menos uma vez, mesmo que a condição seja falsa desde o começo.

Funções e Tratamento de Erros

Definindo e Calling Funções

As funções servem para juntar um bloco de código que faz uma tarefa específica. Assim, em vez de reescrever o mesmo código várias vezes, você só precisa chamar a função. No JavaScript, existem três formas principais de criar uma função:

  • Declaração de função: É o jeito tradicional. Você usa a palavra function, dá um nome para ela, coloca parênteses () e chaves {}.
  • Expressão de função: É quando você cria uma função sem nome (anônima) e a guarda dentro de uma variável.
  • Arrow Functions: É um jeito moderno e mais curto de escrever funções, usando o símbolo de uma seta (=>).
  • Chamando função: Não importa o jeito que você criou a função, para fazer ela funcionar você só precisa digitar o nome dela seguido de parênteses, assim: nomeDaFuncao().

Parâmetros e Argumentos

Para que uma função seja útil em situações diferentes, você pode enviar informações para dentro dela. Parâmetros são os nomes que você coloca dentro dos parênteses na hora de criar a função (eles funcionam como variáveis do lado de dentro). Argumentos são os dados reais que você envia para esses parâmetros na hora de chamar a função.

Valores de Retorno (return)

Enquanto os parâmetros servem para enviar dados para dentro da função, o return serve para enviar um resultado para fora dela.

Atenção:

Assim que o JavaScript encontra a palavra return, a função faz o cálculo, envia o resultado de volta e para de rodar imediatamente. Qualquer código que estiver abaixo do return será ignorado.

Tratamento de Erros

Para evitar que o seu programa trave do nada, você precisa lidar com os erros. No JavaScript, os problemas costumam ser de três tipos: erros de digitação (sintaxe), erros que acontecem enquanto o programa roda (tempo de execução) e erros de lógica (quando o código roda, mas o resultado sai errado).

  • Geração ativa de erros (throw): Se alguma regra do seu sistema for violada, você pode parar o programa manualmente e avisar que algo deu errado usando o comando throw new Error('Mensagem de erro').
  • A estrutura try-catch: É a rede de proteção do código. No bloco try, você coloca o código que tem risco de dar algum erro. Se algo falhar ali dentro, o JavaScript para o try na hora e pula direto para o bloco catch(error). Lá dentro, você decide o que fazer com o erro (como mostrar uma mensagem amigável na tela) para evitar que o site ou aplicativo trave completamente para o usuário.

Objetos e Arrays

Usando Objetos

Objetos são como "caixas" organizadas que guardam informações (variáveis) e ações (funções) que têm a ver com o mesmo assunto.

  • Notação literal de objeto {}: A forma mais simples de criar um objeto é abrindo e fechando chaves {}.
  • Propriedades e Métodos: O que está dentro das chaves são as propriedades (as características do objeto, como cor ou tamanho) e os métodos (as funções que o objeto pode executar, como "correr" ou "ligar").
  • Notação de Ponto (.): Para ver, mudar ou ativar algo dentro do objeto, você usa o nome dele seguido de um ponto e o nome do que quer acessar. Exemplo: objeto.propriedade.
  • Palavra-chave this: Quando você usa this dentro de um método, ele significa "este objeto aqui". Serve para o objeto conseguir mexer nas suas próprias propriedades. Exemplo: this.nome.

Usando Arrays

Arrays (também chamados de listas ou vetores) servem para guardar vários dados de uma vez só, em forma de lista sequencial.

  • Notação literal de array []: Você cria uma lista colocando os dados dentro de colchetes [] e separando cada um por vírgula.
  • Índices Numéricos Posicionais(Posição na lista): O JavaScript começa a contar as posições do array do número 0. O primeiro item fica no índice 0, o segundo no índice 1, e assim por diante. Exemplo: lista[0].
  • Inserção Dinâmica de dados: Você pode colocar um dado em uma posição vazia (ex: cores[3] = 'laranja') ou usar o comando .push(), que joga o novo item direto para o final da lista.
  • Mapeamento e Iteração Cíclica: Para olhar item por item de um array, usamos laços de repetição (como o for). Para o código saber o tamanho exato da lista, usamos o comando .length.

Comandos úteis para Arrays
.concat(): Junta duas listas em uma só.
.filter(): Cria uma nova lista apenas com os itens que passarem em um teste.
.forEach(): Passa por cada item da lista para fazer alguma ação.
.join(): Junta todos os itens da lista e os transforma em um único texto.
.map(): Altera todos os itens da lista de uma vez e cria uma lista nova com os resultados.
.pop(): Apaga o último item da lista.
.reverse(): Inverte a ordem da lista (o que estava no fim vai para o começo).
.shift(): Apaga o primeiro item da lista.
.slice(): Corta e pega um pedaço específico da lista.
.splice(): Tira ou adiciona itens em qualquer posição que você escolher.
.sort(): Organiza a lista em ordem (alfabética ou numérica).
